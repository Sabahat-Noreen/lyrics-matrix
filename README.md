# "I'm a Soldier Born to Die" - Poem Webpage

A minimalist, beautifully designed webpage dedicated to the poem **"I'm a Soldier Born to Die"**. This project serves as a digital tribute, combining the written text with a carefully selected background image and an author profile.

## Project Structure

The project is organized into a main directory containing the configuration files and a dedicated assets folder for the webpage source files.

```text
.
├── README.md               # Project documentation (this file)
└── I'm a Soldier Born to Die         # Main source folder
    ├── index.html          # Webpage structure and poem text
    ├── style.css           # Styling and layout design
    ├── author.jpg          # Profile picture of the author
    └── image.jpg           # Background imagery for the webpage
